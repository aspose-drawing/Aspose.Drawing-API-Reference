## Pen.CompoundArray property

複合ペンを指定する値の配列を取得または設定します。複合ペンは、 平行線とスペースで構成される複合線を描画します.

```csharp
public float[] CompoundArray { get; set; }
```

### 戻り値

複合配列を指定する実数の配列。 配列内の要素は、0 以上 1 以下の昇順でなければなりません。
