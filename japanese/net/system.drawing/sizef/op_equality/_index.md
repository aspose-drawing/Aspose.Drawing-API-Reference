## SizeF Equality operator

2 つのSizeF構造は等しい.

```csharp
public static bool operator ==(SizeF sz1, SizeF sz2)
```

| パラメータ | タイプ | 説明 |
| --- | --- | --- |
| sz1 | SizeF | のSizeF等号演算子の左側の構造。 |
| sz2 | SizeF | のSizeF等号演算子の右側の構造。 |

### 戻り値

この演算子は、次の場合に true を返します。*sz1*と*sz2*幅と高さが等しい。 そうでない場合は false。
