## StringFormat.DigitSubstitutionLanguage property

Ottiene la lingua utilizzata quando le cifre locali vengono sostituite con quelle occidentali.

```csharp
public int DigitSubstitutionLanguage { get; }
```

### Valore di ritorno

Un identificatore di lingua NLS (National Language Support) che identifica la lingua che verrà utilizzata quando le cifre locali verranno sostituite con quelle occidentali. Puoi passare ilLCID proprietà di aCultureInfo oggetto come identificatore di lingua NLS . Ad esempio, supponiamo di creare unCultureInfooggetto passando la stringa "ar-EG" ad aCultureInfo costruttore. Se passi ilLCID proprietà di quelloCultureInfo oggetto insieme a.Traditional a il[`SetDigitSubstitution`](../setdigitsubstitution) metodo, le cifre arabo-indiane verranno sostituite con cifre occidentali al momento della visualizzazione.
