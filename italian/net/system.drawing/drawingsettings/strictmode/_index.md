## DrawingSettings.StrictMode property

Ottiene o imposta un valore che indica se è abilitata la cattura più rigorosa delle funzionalità non implementate.

```csharp
public static bool StrictMode { get; set; }
```

### Osservazioni

Se impostato su true, in caso di utilizzo di funzionalità/parametri che non funzionano correttamente nell'implementazione corrente la libreria genererà NotImplementedException. Se impostato su false, alcuni parametri potrebbero essere ignorati per dare al programma utente la possibilità di funzionare. In tal caso il i risultati del disegno potrebbero avere un aspetto diverso rispetto a GDI+.
