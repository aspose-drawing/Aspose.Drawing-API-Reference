## Pen.CompoundArray property

Ottiene o imposta una matrice di valori che specifica una penna composta. Una penna composta disegna una linea composta composta da linee e spazi paralleli.

```csharp
public float[] CompoundArray { get; set; }
```

### Valore di ritorno

Un array di numeri reali che specifica l'array composto. Gli elementi nell'array devono essere in ordine crescente, non minore di 0 e non maggiore di 1.
