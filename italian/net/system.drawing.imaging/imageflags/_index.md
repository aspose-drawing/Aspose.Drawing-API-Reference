## ImageFlags enumeration

Specifica gli attributi dei dati pixel contenuti in un[`Image`](../../system.drawing/image) oggetto. La proprietà Flags restituisce un membro di questa enumerazione. Questa enumerazione ha un attributo FlagsAttribute che consente una combinazione bit per bit dei valori dei suoi membri.

```csharp
[Flags]
public enum ImageFlags
```

### I valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| None | `0` | Non ci sono informazioni sul formato |
| Scalable | `1` | I dati dei pixel sono scalabili. |
| HasAlpha | `2` | I dati pixel contengono informazioni alfa. |
| HasTranslucent | `4` | Specifica che i dati pixel hanno valori alfa diversi da 0 (trasparente) e 255 (opaco) |
| PartiallyScalable | `8` | I dati dei pixel sono parzialmente scalabili, ma ci sono alcune limitazioni. |
| ColorSpaceRgb | `10` | I dati pixel utilizzano uno spazio colore RGB. |
| ColorSpaceCmyk | `20` | I dati pixel utilizzano uno spazio colore CMYK. |
| ColorSpaceGray | `40` | I dati pixel sono in scala di grigi. |
| ColorSpaceYcbcr | `80` | Specifica che l'immagine viene archiviata utilizzando uno spazio colore YCBCR. |
| ColorSpaceYcck | `100` | Specifica che l'immagine viene archiviata utilizzando uno spazio colore YCCK. |
| HasRealDpi | `1000` | Specifica che le informazioni sui punti per pollice vengono archiviate nell'immagine. |
| HasRealPixelSize | `2000` | Specifica che la dimensione dei pixel è memorizzata nell'immagine. |
| ReadOnly | `10000` | I dati del pixel sono di sola lettura. |
| Caching | `20000` | I dati dei pixel possono essere memorizzati nella cache per un accesso più rapido. |
