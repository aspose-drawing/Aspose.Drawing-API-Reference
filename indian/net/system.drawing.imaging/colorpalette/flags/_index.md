## ColorPalette.Flags property

```csharp
public int Flags { get; }
```

### प्रतिलाभ की मात्रा

निम्नलिखित ध्वज मान मान्य हैं: 0x00000001 सरणी में रंग मानों में अल्फा जानकारी होती है। 0x00000002 सरणी में रंग ग्रेस्केल मान हैं।
