## BitmapData class

Καθορίζει τα χαρακτηριστικά μιας εικόνας bitmap. οBitmapData η κλάση χρησιμοποιείται από το LockBits καιUnlockBits μέθοδοι τουBitmap τάξη. Δεν κληρονομείται.

```csharp
public sealed class BitmapData
```
