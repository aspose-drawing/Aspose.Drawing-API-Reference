## FillPie(Brush, Rectangle, float, float) {#fillpie_2}

Γεμίζει το εσωτερικό ενός τμήματος πίτας που ορίζεται από μια έλλειψη που καθορίζεται από μια ορθογώνια δομή και δύο ακτινικές γραμμές.

```csharp
public void FillPie(Brush brush, Rectangle rect, float startAngle, float sweepAngle)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |
| brush | Brush | Βούρτσα που καθορίζει τα χαρακτηριστικά του γεμίσματος. |
| rect | Rectangle | Ορθογώνια δομή που αντιπροσωπεύει το οριοθετημένο ορθογώνιο που ορίζει την έλλειψη από την οποία προέρχεται το τμήμα της πίτας. |
| startAngle | Single | Η γωνία σε μοίρες μετράται δεξιόστροφα από τον άξονα x στην πρώτη πλευρά του τμήματος της πίτας. |
| sweepAngle | Single | Η γωνία σε μοίρες μετράται δεξιόστροφα από την παράμετρο startAngle στη δεύτερη πλευρά του τμήματος πίτας. |
