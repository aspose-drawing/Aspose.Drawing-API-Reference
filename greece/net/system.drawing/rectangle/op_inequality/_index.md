## Rectangle Inequality operator

```csharp
public static bool operator !=(Rectangle left, Rectangle right)
```

| Παράμετρος | Τύπος | Περιγραφή |
| --- | --- | --- |

| right | Rectangle | οRectangle δομή που βρίσκεται στα δεξιά του τελεστή ανισότητας. |
