## MetafileType enumeration

Gibt Typen von Metadateien an.

```csharp
public enum MetafileType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Invalid | `0` | Gibt ein Metadateiformat an, das in GDI+ nicht erkannt wird. |
| Wmf | `1` | Gibt eine WMF-Datei (Windows Metafile) an. Eine solche Datei enthält nur GDI-Einträge. |
| WmfPlaceable | `2` | Gibt eine WMF-Datei (Windows Metafile) an, der ein platzierbarer Metadatei-Header vorangestellt ist. |
| Emf | `3` | Gibt eine Enhanced Metafile (EMF)-Datei an. Eine solche Datei enthält nur GDI-Einträge. |
| EmfPlusOnly | `4` | Gibt eine EMF+-Datei an. Eine solche Datei enthält nur GDI+-Datensätze und muss mithilfe von GDI+ angezeigt werden. Das Anzeigen der Datensätze mit GDI kann zu unvorhersehbaren Ergebnissen führen. |
| EmfPlusDual | `5` | Gibt eine EMF+ Dual-Datei an. Eine solche Datei enthält GDI+-Datensätze zusammen mit alternativen GDI-Datensätzen und kann entweder mit GDI oder GDI+ angezeigt werden. Die Anzeige der Datensätze mit GDI kann zu Qualitätseinbußen führen. |
