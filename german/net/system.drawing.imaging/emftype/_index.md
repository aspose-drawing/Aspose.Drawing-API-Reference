## EmfType enumeration

Gibt die Art der Datensätze an, die in einer EMF-Datei (Enhanced Metafile) abgelegt werden. Diese Aufzählung wird von mehreren Konstruktoren in der verwendetMetafile Klasse.

```csharp
public enum EmfType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| EmfOnly | `3` | Gibt an, dass alle Datensätze in der Metadatei EMF-Datensätze sind, die von GDI oder GDI+ angezeigt werden können. |
| EmfPlusOnly | `4` | Gibt an, dass alle Datensätze in der Metadatei EMF+-Datensätze sind, die von GDI+, aber nicht von GDI angezeigt werden können. |
| EmfPlusDual | `5` | Gibt an, dass alle EMF+-Datensätze in der Metadatei einem alternativen EMF-Datensatz zugeordnet sind. Metafiles of typeEmfPlusDual kann von GDI oder von GDI+ angezeigt werden. |
