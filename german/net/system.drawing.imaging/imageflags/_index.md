## ImageFlags enumeration

Gibt die Attribute der Pixeldaten an, die in einer enthalten sind[`Image`](../../system.drawing/image) Objekt. Die Flags-Eigenschaft gibt ein Mitglied dieser Aufzählung zurück. Diese Aufzählung hat ein FlagsAttribute-Attribut, das eine bitweise Kombination ihrer Mitgliedswerte ermöglicht.

public enum ImageFlags

| Name | Wert | Beschreibung |
| --- | --- | --- |
| None | `0` | Es gibt keine Formatinformationen |
| Scalable | `1` | Die Pixeldaten sind skalierbar. |
| HasAlpha | `2` | Die Pixeldaten enthalten Alpha-Informationen. |
| HasTranslucent | `4` | Gibt an, dass die Pixeldaten andere Alphawerte als 0 (transparent) und 255 (deckend) haben |
