## TextRenderingHint enumeration

Gibt die Qualität der Textwiedergabe an.

```csharp
public enum TextRenderingHint
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| SystemDefault | `0` | Jedes Zeichen wird mit seiner Glyphen-Bitmap gezeichnet, mit dem standardmäßigen Rendering-Hinweis des Systems. Der Text wird mit den Einstellungen zur Schriftglättung gezeichnet, die der Benutzer für das System ausgewählt hat. |
| SingleBitPerPixelGridFit | `1` | Jedes Zeichen wird mit seiner Glyphen-Bitmap gezeichnet. Hinting wird verwendet, um das Erscheinungsbild des Zeichens auf Stielen und Krümmungen zu verbessern. |
| SingleBitPerPixel | `2` | Jedes Zeichen wird mit seiner Glyphen-Bitmap gezeichnet. Hinweise werden nicht verwendet. |
| AntiAliasGridFit | `3` | Jedes Zeichen wird mit seiner geglätteten Glyphen-Bitmap mit Hinweisen gezeichnet. Viel bessere Qualität aufgrund von Antialiasing, aber mit höheren Leistungskosten. |
| AntiAlias | `4` | Jedes Zeichen wird mit seiner geglätteten Glyphen-Bitmap ohne Hinting gezeichnet. Bessere Qualität durch Antialiasing. Unterschiede in der Stammbreite können erkennbar sein, da Hinting deaktiviert ist. |
| ClearTypeGridFit | `5` | Jedes Zeichen wird mit seiner Glyphe gezeichnet ClearType-Bitmap mit Hinting. Die höchste Qualitätseinstellung. Wird verwendet, um die Vorteile von ClearType-Schriftartenfunktionen zu nutzen. |
