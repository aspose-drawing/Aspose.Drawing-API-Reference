## CoordinateSpace enumeration

Gibt das System an, das beim Auswerten von Koordinaten verwendet werden soll.

```csharp
public enum CoordinateSpace
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| World | `0` | Gibt an, dass sich Koordinaten im Weltkoordinatenkontext befinden. Weltkoordinaten werden in einer nichtphysischen Umgebung verwendet, beispielsweise in einer Modellierungsumgebung. |
| Page | `1` | Gibt an, dass sich Koordinaten im Seitenkoordinatenkontext befinden. Ihre Einheiten sind durch die Graphics.PageUnit-Eigenschaft definiert und müssen eines der Elemente der GraphicsUnit-Enumeration sein. |
| Device | `2` | Gibt an, dass sich Koordinaten im Gerätekoordinatenkontext befinden. Auf einem Computerbildschirm werden die Gerätekoordinaten normalerweise in Pixeln gemessen. |
