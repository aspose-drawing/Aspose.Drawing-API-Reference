## GraphicsPathIterator.NextPathType method

Ruft den Startindex und den Endindex der nächsten Gruppe von Datenpunkten ab, die alle denselben Typ haben.

```csharp
public int NextPathType(out byte pathType, out int startIndex, out int endIndex)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| pathType | Byte& | [out] Erhält den Punkttyp, den alle Punkte in der Gruppe gemeinsam haben. Mögliche Typen können aus der abgerufen werdenPathPointType Aufzählung. |
| startIndex | Int32& | [out] Erhält den Anfangsindex der Punktegruppe. |
| endIndex | Int32& | [out] Erhält den Endindex der Punktegruppe. |

### Rückgabewert

Diese Methode gibt die Anzahl der Datenpunkte in der Gruppe zurück. Wenn es keine weiteren Gruppen im Pfad gibt, gibt diese Methode 0 zurück.
