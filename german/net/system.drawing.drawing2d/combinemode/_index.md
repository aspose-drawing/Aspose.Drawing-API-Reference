## CombineMode enumeration

Gibt an, wie verschiedene Schnittbereiche kombiniert werden können.

```csharp
public enum CombineMode
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Replace | `0` | Ein Clipping-Bereich wird durch einen anderen ersetzt. |
| Intersect | `1` | Zwei Clipping-Regionen werden kombiniert, indem ihr Schnittpunkt genommen wird. |
| Union | `2` | Zwei Clipping-Regionen werden kombiniert, indem beide zusammengeführt werden. |
| Xor | `3` | Zwei Clipping-Regionen werden kombiniert, indem nur die Bereiche genommen werden, die von der einen oder anderen Region umschlossen sind, aber nicht beide. |
| Exclude | `4` | Gibt an, dass die vorhandene Region durch das Ergebnis der Entfernung der neuen Region aus der vorhandenen Region ersetzt wird. Anders gesagt, die neue Region wird aus der vorhandenen Region ausgeschlossen. |
| Complement | `5` | Gibt an, dass die vorhandene Region durch das Ergebnis der Entfernung der vorhandenen Region aus der neuen Region ersetzt wird. Anders gesagt, die vorhandene Region wird aus der neuen Region ausgeschlossen. |
