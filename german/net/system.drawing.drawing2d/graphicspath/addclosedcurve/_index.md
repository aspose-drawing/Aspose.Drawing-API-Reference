## AddClosedCurve(PointF[], float) {#addclosedcurve_1}

Fügt diesem Pfad eine geschlossene Kurve hinzu. Es wird eine kardinale Spline-Kurve verwendet, da die Kurve durch jeden der Punkte im Array verläuft.

```csharp
public void AddClosedCurve(PointF[] points, float tension)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| points | PointF[] | Ein Array von PointF-Strukturen, das die Punkte darstellt, die die Kurve definieren. |
| tension | Single | Ein Wert zwischen 0 und 1, der den Betrag angibt, um den sich die Kurve zwischen Punkten krümmt , wobei 0 die kleinste Kurve (schärfste Ecke) und 1 die glatteste Kurve ist. |
