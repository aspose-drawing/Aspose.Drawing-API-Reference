## FlushIntention enumeration

Gibt an, ob Befehle im Grafikstapel sofort beendet (geflusht) oder so schnell wie möglich ausgeführt werden.

```csharp
public enum FlushIntention
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Flush | `0` | Gibt an, dass der Stack aller Grafikoperationen sofort geleert wird. |
| Sync | `1` | Gibt an, dass alle Grafikoperationen auf dem Stack so schnell wie möglich ausgeführt werden. Dadurch wird der Grafikstatus synchronisiert. |
