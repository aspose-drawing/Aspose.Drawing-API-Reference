Füllt das Innere eines Tortenabschnitts, der durch eine Ellipse definiert ist, die durch eine Rectangle-Struktur und zwei radiale Linien definiert ist.

```csharp
public void FillPie(Brush brush, Rectangle rect, float startAngle, float sweepAngle)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| brush | Brush | Pinsel, der die Eigenschaften der Füllung bestimmt. |
| rect | Rectangle | Rectangle-Struktur, die das umgebende Rechteck darstellt, das die Ellipse definiert, aus der der Kreisabschnitt stammt. |
| startAngle | Single | Winkel in Grad, gemessen im Uhrzeigersinn von der x-Achse zur ersten Seite des Tortenabschnitts. |
| sweepAngle | Single | Winkel in Grad, gemessen im Uhrzeigersinn vom startAngle-Parameter zur zweiten Seite des Tortenabschnitts. |
