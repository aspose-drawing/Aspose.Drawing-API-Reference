## DrawClosedCurve(Pen, PointF[], float, FillMode) {#drawclosedcurve_1}

Zeichnet einen geschlossenen kardinalen Spline, der durch ein Array von PointF-Strukturen mit einer angegebenen Spannung definiert ist.

```csharp
public void DrawClosedCurve(Pen pen, PointF[] points, float tension, FillMode fillmode)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| pen | Pen | Pen die die Farbe, Breite und Höhe der Kurve bestimmt. |
| points | PointF[] | Anordnung vonPointF Strukturen, die den Spline definieren. |
| tension | Single | Wert größer oder gleich 0,0 F, der die Spannung der Kurve angibt. |
| fillmode | FillMode | Mitglied der FillMode-Enumeration, die bestimmt, wie die Kurve gefüllt wird. Dieser Parameter ist erforderlich, wird aber ignoriert. |
