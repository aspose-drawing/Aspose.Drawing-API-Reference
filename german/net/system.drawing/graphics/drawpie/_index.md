## DrawPie(Pen, RectangleF, float, float) {#drawpie_1}

Zeichnet eine Tortenform, die durch eine Ellipse definiert ist, die durch eine RectangleF-Struktur und zwei radiale Linien angegeben wird.

```csharp
public void DrawPie(Pen pen, RectangleF rect, float startAngle, float sweepAngle)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| pen | Pen | Stift, der Farbe, Breite und Stil der Tortenform bestimmt. |
| rect | RectangleF | RectangleF-Struktur, die das umgebende Rechteck darstellt, das die Ellipse definiert, aus der die Tortenform stammt. |
| startAngle | Single | Winkel gemessen in Grad im Uhrzeigersinn von der x-Achse zur ersten Seite der Tortenform. |
| sweepAngle | Single | Winkel gemessen in Grad im Uhrzeigersinn vom Parameter startAngle zur zweiten Seite der Tortenform. |
