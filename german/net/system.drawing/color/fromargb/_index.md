## FromArgb(int, int, int) {#fromargb_1}

Erstellt einColor Struktur aus den angegebenen 8-Bit-Farbwerten (Rot, Grün und Blau). Der Alpha-Wert ist implizit 255 (vollständig deckend). Obwohl diese Methode die Übergabe eines 32-Bit-Werts für jede Farbkomponente zulässt, die Wert jeder Komponente ist auf 8 Bit begrenzt.

```csharp
public static Color FromArgb(int red, int green, int blue)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| red | Int32 | Der rote Komponentenwert für das NeueColor. Gültige Werte sind 0 bis 255. |
| green | Int32 | Der grüne Komponentenwert für den NeuenColor. Gültige Werte sind 0 bis 255. |
| blue | Int32 | Der blaue Komponentenwert für das NeueColor. Gültige Werte sind 0 bis 255. |
