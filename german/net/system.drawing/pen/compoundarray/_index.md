## Pen.CompoundArray property

Ruft ein Array von Werten ab oder legt es fest, das einen zusammengesetzten Stift angibt. Ein zusammengesetzter Stift zeichnet eine zusammengesetzte Linie, die aus parallelen Linien und Zwischenräumen besteht.

```csharp
public float[] CompoundArray { get; set; }
```

### Rückgabewert

Ein Array aus reellen Zahlen, das das zusammengesetzte Array angibt. Die Elemente im Array müssen in aufsteigender Reihenfolge, nicht kleiner als 0 und nicht größer als 1 sein.
