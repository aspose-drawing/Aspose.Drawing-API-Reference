## Graphics.DrawImageAbort delegate

Stellt eine Rückrufmethode bereit, um zu entscheiden, wann die[`DrawImage`](../graphics/drawimage) -Methode sollte die Ausführung vorzeitig abbrechen und das Zeichnen eines Bildes beenden.

```csharp
public delegate bool DrawImageAbort(IntPtr callbackdata);
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| callbackdata | IntPtr | Interner Zeiger, der Daten für die Callback-Methode angibt. Dieser Parameter wird nicht von allen übergeben[`DrawImage`](../graphics/drawimage) Überlastungen. Sie können das Fehlen testen, indem Sie nach dem Wert suchenZero . |

### Rückgabewert

Diese Methode gibt true zurück, wenn sie entscheidet, dass die[`DrawImage`](../graphics/drawimage) -Methode sollte die Ausführung vorzeitig beenden. Andernfalls wird false zurückgegeben, um anzuzeigen, dass die[`DrawImage`](../graphics/drawimage) Methode sollte die Ausführung fortsetzen.
