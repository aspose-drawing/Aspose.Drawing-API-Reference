## StringFormat.DigitSubstitutionLanguage property

Ruft die Sprache ab, die verwendet wird, wenn lokale Ziffern durch westliche Ziffern ersetzt werden.

```csharp
public int DigitSubstitutionLanguage { get; }
```

### Rückgabewert

Eine National Language Support (NLS)-Sprachkennung, die die Sprache identifiziert, die verwendet wird, wenn lokale Ziffern durch westliche Ziffern ersetzt werden. Sie können die passierenLCID Eigentum von aCultureInfo Objekt als NLS-Sprachkennung. Angenommen, Sie erstellen beispielsweise eineCultureInfoObjekt, indem Sie den String "ar-EG" an a übergebenCultureInfo Konstrukteur. Wenn Sie die bestehenLCID property davonCultureInfo Objekt zusammen mit.Traditional to die[`SetDigitSubstitution`](../setdigitsubstitution) -Methode, dann werden zur Anzeigezeit westliche Ziffern durch arabisch-indische Ziffern ersetzt.
