## Rectangle Inequality operator

Testet ob zweiRectangle Strukturen unterscheiden sich in Lage oder Größe.

```csharp
public static bool operator !=(Rectangle left, Rectangle right)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| left | Rectangle | DasRectangle Struktur, die sich links vom Ungleichheitsoperator befindet. |
| right | Rectangle | DasRectangle Struktur, die rechts vom Ungleichheitsoperator steht. |

### Rückgabewert

Dieser Operator kehrt zurück`Stimmt` wenn eine der X-, Y-, Breiten- oder Höheneigenschaften der beidenRectangle Strukturen sind ungleich; Andernfalls`FALSCH` .
