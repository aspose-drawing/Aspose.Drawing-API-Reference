## StringAlignment enumeration

Gibt die Ausrichtung einer Textzeichenfolge relativ zu ihrem Layoutrechteck an.

```csharp
public enum StringAlignment
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Near | `0` | Gibt an, dass der Text in der Nähe des Layouts ausgerichtet werden soll. Bei einem von links nach rechts verlaufenden Layout ist die Nahposition links. Bei einem Rechts-nach-links-Layout ist die Nahposition rechts. |
| Center | `1` | Gibt an, dass Text in der Mitte des Layoutrechtecks ausgerichtet wird. |
| Far | `2` | Gibt an, dass der Text weit entfernt von der Ursprungsposition des Layoutrechtecks ausgerichtet wird. In einem Layout von links nach rechts ist die äußerste Position rechts. In einem Layout von rechts nach links ist die äußerste Position links. |
