## PixelFormat enumeration

Especifica el formato de los datos de color para cada píxel de la imagen.

public enum PixelFormat

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Indexed | `65536` | Los datos de píxeles contienen valores indexados por color, lo que significa que los valores son un índice de los colores en la tabla de colores del sistema , a diferencia de los valores de color individuales. |
| Gdi | `131072` | Los datos de píxeles contienen colores GDI. |
| Alpha | `262144` | Los datos de píxeles contienen valores alfa que no están premultiplicados. |

| DontCare | `0` | No se especifica ningún formato de píxel. |
| Format1bppIndexed | `196865` | Especifica que el formato de píxel es de 1 bit por píxel y que utiliza color indexado. Por lo tanto, la tabla de colores tiene dos colores. |

| Format16bppArgb1555 | `397319` | El formato de píxel es de 16 bits por píxel. La información de color especifica 32.768 tonos de color, de los cuales 5 bits son rojos, 5 bits son verdes, 5 bits son azules y 1 bit es alfa. |

| Format48bppRgb | `1060876` | Especifica que el formato es de 48 bits por píxel; Se utilizan 16 bits cada uno para los componentes rojo, verde y azul. |
