## MetafileType enumeration

Especifica tipos de metarchivos.

```csharp
public enum MetafileType
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Invalid | `0` | Especifica un formato de metarchivo que no se reconoce en GDI+. |
| Wmf | `1` | Especifica un archivo WMF (metarchivo de Windows). Tal archivo contiene solo registros GDI. |
| WmfPlaceable | `2` | Especifica un archivo WMF (metarchivo de Windows) que tiene un encabezado de metarchivo que se puede colocar delante. |
| Emf | `3` | Especifica un archivo de metarchivo mejorado (EMF). Tal archivo contiene solo registros GDI. |
| EmfPlusOnly | `4` | Especifica un archivo EMF+. Dicho archivo contiene solo registros GDI+ y debe visualizarse mediante GDI+. Mostrar los registros usando GDI puede causar resultados impredecibles. |
| EmfPlusDual | `5` | Especifica un archivo EMF+ Dual. Un archivo de este tipo contiene registros GDI+ junto con registros GDI alternativos y se puede mostrar mediante GDI o GDI+. Mostrar los registros mediante GDI puede causar cierta degradación de la calidad. |
