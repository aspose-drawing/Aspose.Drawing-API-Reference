## DrawingSettings.StrictMode property

Obtiene o establece un valor que indica si está habilitada la captura más estricta de características no implementadas.

```csharp
public static bool StrictMode { get; set; }
```

### Observaciones

Si se establece en verdadero, en caso de usar características/parámetros que no funcionan correctamente en la implementación actual la biblioteca lanzará NotImplementedException. Si se establece en falso, algunos parámetros pueden ignorarse para dar oportunidad de trabajo al programa de usuario. En ese caso, el los resultados del dibujo pueden verse diferentes en comparación con GDI+.
