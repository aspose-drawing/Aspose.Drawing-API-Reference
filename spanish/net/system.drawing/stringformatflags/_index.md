## StringFormatFlags enumeration

Especifica la información de visualización y diseño para cadenas de texto.

```csharp
[Flags]
public enum StringFormatFlags
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| DirectionRightToLeft | `1` | El texto se muestra de derecha a izquierda. |
| DirectionVertical | `2` | El texto está alineado verticalmente. |
| FitBlackBox | `4` | Se permite que partes de los caracteres sobresalgan del rectángulo de diseño de la cadena. De forma predeterminada, los caracteres se reposicionan para evitar que sobresalgan. |
| DisplayFormatControl | `20` | Los caracteres de control, como la marca de izquierda a derecha, se muestran en la salida con un glifo representativo. |
| NoFontFallback | `400` | El recurso a fuentes alternativas para los caracteres no admitidos en la fuente solicitada está deshabilitado. Los caracteres faltantes se muestran con el glifo faltante de las fuentes, generalmente un cuadrado abierto. |
| MeasureTrailingSpaces | `800` | Incluye el espacio final al final de cada línea. De manera predeterminada, el rectángulo de límite devuelto por el método MeasureString excluye el espacio al final de cada línea. Establezca esta bandera para incluir ese espacio en la medición. |
| NoWrap | `1000` | Ajuste de texto entre líneas cuando el formato dentro de un rectángulo está deshabilitado. Esta bandera está implícita cuando se pasa un punto en lugar de un rectángulo, o cuando el rectángulo especificado tiene una longitud de línea cero. |
| LineLimit | `2000` | Solo las líneas completas se disponen en el rectángulo de formato. De forma predeterminada, el diseño continúa hasta el final del texto, o hasta que no se ven más líneas como resultado del recorte, lo que suceda primero. Tenga en cuenta que la configuración predeterminada permite la última línea se oscurecerá parcialmente con un formato de rectángulo que no es un múltiplo entero de la altura de la línea. Para garantizar que solo se vean líneas completas, especifique este valor y tenga cuidado de proporcionar un rectángulo de formato al menos tan alto como la altura de una línea. |
| NoClip | `4000` | Se permite mostrar las partes sobresalientes de los glifos y el texto sin ajustar que sobresale del rectángulo de formato. De manera predeterminada, se recortan todas las partes del texto y de los glifos que sobresalen del rectángulo de formato. |
