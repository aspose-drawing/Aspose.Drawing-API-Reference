## StringFormat.DigitSubstitutionLanguage property

Obtiene el idioma que se utiliza cuando los dígitos locales se sustituyen por dígitos occidentales.

```csharp
public int DigitSubstitutionLanguage { get; }
```

### Valor_devuelto

Un identificador de idioma compatible con el idioma nacional (NLS) que identifica el idioma que se utilizará cuando los dígitos locales se sustituyan por dígitos occidentales. Puedes pasar elLCID propiedad de unCultureInfo objeto como el identificador de idioma NLS. Por ejemplo, suponga que crea unCultureInfoobjeto pasando la cadena "ar-EG" a unCultureInfo constructor. Si pasas elLCID property de esoCultureInfo objeto junto con.Traditional to el[`SetDigitSubstitution`](../setdigitsubstitution) método, los dígitos árabe-índicos se sustituirán por dígitos occidentales en el momento de visualización.
