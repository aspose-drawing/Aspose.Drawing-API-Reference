## FlushIntention enumeration

Spécifie si les commandes de la pile graphique sont terminées (vidées) immédiatement ou exécutées dès que possible.

```csharp
public enum FlushIntention
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| Flush | `0` | Spécifie que la pile de toutes les opérations graphiques est vidée immédiatement. |
| Sync | `1` | Spécifie que toutes les opérations graphiques sur la pile sont exécutées dès que possible. Cela synchronise l'état des graphiques. |
