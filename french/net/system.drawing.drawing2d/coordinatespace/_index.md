## CoordinateSpace enumeration

Spécifie le système à utiliser lors de l'évaluation des coordonnées.

```csharp
public enum CoordinateSpace
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| World | `0` | Spécifie que les coordonnées sont dans le contexte des coordonnées universelles. Les coordonnées mondiales sont utilisées dans un environnement non physique, tel qu'un environnement de modélisation. |
| Page | `1` | Spécifie que les coordonnées sont dans le contexte des coordonnées de la page. Leurs unités sont définies par la propriété Graphics.PageUnit et doivent être l'un des éléments de l'énumération GraphicsUnit. |
| Device | `2` | Spécifie que les coordonnées sont dans le contexte des coordonnées du périphérique. Sur un écran d'ordinateur , les coordonnées de l'appareil sont généralement mesurées en pixels. |
