## CompositingMode enumeration

Spécifie comment les couleurs source sont combinées avec les couleurs d'arrière-plan.

```csharp
public enum CompositingMode
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| SourceOver | `0` | Spécifie que lorsqu'une couleur est rendue, elle est mélangée avec la couleur d'arrière-plan. Le mélange est déterminé par le composant alpha de la couleur rendue. |
| SourceCopy | `1` | Spécifie que lorsqu'une couleur est rendue, elle écrase la couleur d'arrière-plan. |
