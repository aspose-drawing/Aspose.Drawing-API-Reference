## StringFormat.DigitSubstitutionLanguage property

Obtient la langue utilisée lorsque des chiffres locaux sont remplacés par des chiffres occidentaux.

```csharp
public int DigitSubstitutionLanguage { get; }
```

### Return_Value

Un identifiant de langue NLS (National Language Support) qui identifie la langue qui sera utilisée lorsque des chiffres locaux seront remplacés par des chiffres occidentaux. Vous pouvez passer leLCID propriété d'unCultureInfo objet comme identifiant de langue NLS. Par exemple, supposons que vous créez unCultureInfoobjet en passant la chaîne "ar-EG" à unCultureInfo constructeur. Si vous passez leLCID propriété de celle-ciCultureInfo objet avec.Traditional à le[`SetDigitSubstitution`](../setdigitsubstitution) méthode, alors les chiffres arabo-indiens seront remplacés par chiffres occidentaux au moment de l'affichage.
