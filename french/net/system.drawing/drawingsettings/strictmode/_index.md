## DrawingSettings.StrictMode property

Obtient ou définit une valeur indiquant si la capture plus stricte des fonctionnalités non implémentées est activée.

```csharp
public static bool StrictMode { get; set; }
```

### Remarques

Si défini sur true, en cas d'utilisation de fonctionnalités/paramètres qui ne fonctionnent pas correctement dans l'implémentation actuelle la bibliothèque lèvera NotImplementedException. Si défini sur false, certains paramètres peuvent être ignorés pour donner au programme utilisateur une chance de fonctionner. Dans ce cas, le les résultats de dessin peuvent sembler différents par rapport à GDI+.
