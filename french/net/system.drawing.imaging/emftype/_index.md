## EmfType enumeration

Spécifie la nature des enregistrements qui sont placés dans un fichier Enhanced Metafile (EMF). Cette énumération est utilisée par plusieurs constructeurs dans leMetafile classe.

```csharp
public enum EmfType
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| EmfOnly | `3` | Spécifie que tous les enregistrements du métafichier sont des enregistrements EMF, qui peuvent être affichés par GDI ou GDI+. |
| EmfPlusOnly | `4` | Spécifie que tous les enregistrements du métafichier sont des enregistrements EMF+, qui peuvent être affichés par GDI+ mais pas par GDI. |
| EmfPlusDual | `5` | Spécifie que tous les enregistrements EMF+ du métafichier sont associés à un autre enregistrement EMF. Métafichiers de typeEmfPlusDual peut être affiché par GDI ou par GDI+. |
