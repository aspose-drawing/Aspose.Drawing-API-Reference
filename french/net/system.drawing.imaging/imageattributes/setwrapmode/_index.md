## SetWrapMode(WrapMode) {#setwrapmode}

Définit le mode d'habillage qui est utilisé pour décider comment disposer une texture en mosaïque sur une forme ou aux limites de la forme. Une texture est en mosaïque sur une forme pour la remplir lorsque la texture est plus petite que la forme qu'elle remplit.

```csharp
public void SetWrapMode(WrapMode mode)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| mode | WrapMode | Un élément deWrapMode qui spécifie comment des copies répétées d'un image sont utilisées pour carreler une zone. |
