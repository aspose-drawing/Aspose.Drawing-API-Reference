## ImageLockMode enumeration

Spécifie les drapeaux qui sont passés au paramètre flags du[`LockBits`](../../system.drawing/bitmap/lockbits) méthode. La[`LockBits`](../../system.drawing/bitmap/lockbits) verrouille une partie d'une image afin que vous puissiez lire ou écrire les données de pixel.

```csharp
public enum ImageLockMode
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| ReadOnly | `1` | Spécifie qu'une partie de l'image est verrouillée pour la lecture. |
| WriteOnly | `2` | Spécifie qu'une partie de l'image est verrouillée en écriture. |
| ReadWrite | `3` | Spécifie qu'une partie de l'image est verrouillée en lecture ou en écriture. |
| UserInputBuffer | `4` | Spécifie que le tampon utilisé pour lire ou écrire des données de pixel est alloué par l'utilisateur. Si cet indicateur est défini, le*flags* paramètre de la[`LockBits`](../../system.drawing/bitmap/lockbits) La méthode sert de paramètre d'entrée (et éventuellement de paramètre de sortie). Si cet indicateur est effacé, alors la*flags* paramètre sert uniquement de paramètre de sortie. |
