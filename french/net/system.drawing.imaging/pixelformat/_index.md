## PixelFormat enumeration

Spécifie le format des données de couleur pour chaque pixel de l'image.

```csharp
public enum PixelFormat
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| Indexed | `65536` | Les données de pixel contiennent des valeurs indexées par couleur, ce qui signifie que les valeurs sont un index des couleurs dans la table de couleurs système, par opposition aux valeurs de couleur individuelles. |
| Gdi | `131072` | Les données de pixel contiennent des couleurs GDI. |
| Alpha | `262144` | Les données de pixel contiennent des valeurs alpha qui ne sont pas prémultipliées. |
| PAlpha | `524288` | Le format pixel contient des valeurs alpha prémultipliées. |
| Extended | `1048576` | Valeur réservée. |
| Canonical | `2097152` | Le format de pixel par défaut de 32 bits par pixel. Le format spécifie une profondeur de couleur de 24 bits et un canal alpha de 8 bits. |
| Undefined | `0` | Le format de pixel n'est pas défini. |
| DontCare | `0` | Aucun format de pixel n'est spécifié. |
| Format1bppIndexed | `196865` | Spécifie que le format pixel est de 1 bit par pixel et qu'il utilise une couleur indexée. La table des couleurs contient donc deux couleurs. |
| Format4bppIndexed | `197634` | Spécifie que le format est de 4 bits par pixel, indexé. |
| Format8bppIndexed | `198659` | Spécifie que le format est de 8 bits par pixel, indexé. La table des couleurs contient donc 256 couleurs. |
| Format16bppGrayScale | `1052676` | Le format de pixel est de 16 bits par pixel. Les informations de couleur spécifient 65536 nuances de gris. |
| Format16bppRgb555 | `135173` | Spécifie que le format est de 16 bits par pixel ; 5 bits chacun sont utilisés pour les composants rouge, vert et bleu. Le bit restant n'est pas utilisé. |
| Format16bppRgb565 | `135174` | Spécifie que le format est de 16 bits par pixel ; 5 bits sont utilisés pour la composante rouge, 6 bits sont utilisés pour la composante verte et 5 bits sont utilisés pour la composante bleue. |
| Format16bppArgb1555 | `397319` | Le format pixel est de 16 bits par pixel. Les informations de couleur spécifient 32 768 nuances de couleur, dont 5 bits sont rouges, 5 bits sont verts, 5 bits sont bleus et 1 bit est alpha. |
| Format24bppRgb | `137224` | Spécifie que le format est de 24 bits par pixel ; 8 bits chacun sont utilisés pour les composants rouge, vert et bleu. |
| Format32bppRgb | `139273` | Spécifie que le format est de 32 bits par pixel ; 8 bits chacun sont utilisés pour les composants rouge, vert et bleu. Les 8 bits restants ne sont pas utilisés. |
| Format32bppArgb | `2498570` | Spécifie que le format est de 32 bits par pixel ; 8 bits chacun sont utilisés pour les composants alpha, rouge, vert et bleu. |
| Format32bppPArgb | `925707` | Spécifie que le format est de 32 bits par pixel ; 8 bits chacun sont utilisés pour les composants alpha, rouge, vert et bleu. Les composantes rouge, verte et bleue sont prémultipliées, selon la composante alpha. |
| Format48bppRgb | `1060876` | Spécifie que le format est de 48 bits par pixel ; 16 bits chacun sont utilisés pour les composants rouge, vert et bleu. |
| Format64bppArgb | `3424269` | Spécifie que le format est de 64 bits par pixel ; 16 bits chacun sont utilisés pour les composants alpha, rouge, vert et bleu. |
| Format64bppPArgb | `1851406` | Spécifie que le format est de 64 bits par pixel ; 16 bits chacun sont utilisés pour les composants alpha, rouge, vert et bleu. Les composantes rouge, verte et bleue sont prémultipliées selon la composante alpha. |
