## ImageFlags enumeration

Spécifie les attributs des données de pixel contenues dans un[`Image`](../../system.drawing/image) objet. La propriété Flags renvoie un membre de cette énumération. Cette énumération a un attribut FlagsAttribute qui permet une combinaison au niveau du bit de ses valeurs de membre.

```csharp
[Flags]
public enum ImageFlags
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| None | `0` | Il n'y a pas d'informations de format |
| Scalable | `1` | Les données de pixel sont évolutives. |
| HasAlpha | `2` | Les données de pixel contiennent des informations alpha. |
| HasTranslucent | `4` | Spécifie que les données de pixel ont des valeurs alpha autres que 0 (transparent) et 255 (opaque) |
| PartiallyScalable | `8` | Les données de pixel sont partiellement évolutives, mais il existe certaines limitations. |
| ColorSpaceRgb | `10` | Les données de pixel utilisent un espace colorimétrique RVB. |
| ColorSpaceCmyk | `20` | Les données de pixel utilisent un espace colorimétrique CMJN. |
| ColorSpaceGray | `40` | Les données de pixel sont en niveaux de gris. |
| ColorSpaceYcbcr | `80` | Spécifie que l'image est stockée à l'aide d'un espace colorimétrique YCBCR. |
| ColorSpaceYcck | `100` | Spécifie que l'image est stockée à l'aide d'un espace colorimétrique YCCK. |
| HasRealDpi | `1000` | Spécifie que les informations de points par pouce sont stockées dans l'image. |
| HasRealPixelSize | `2000` | Spécifie que la taille en pixels est stockée dans l'image. |
| ReadOnly | `10000` | Les données de pixel sont en lecture seule. |
| Caching | `20000` | Les données de pixel peuvent être mises en cache pour un accès plus rapide. |
