## DrawClosedCurve(Pen, PointF[], float, FillMode) {#drawclosedcurve_1}

지정된 장력을 사용하여 PointF 구조 배열로 정의된 닫힌 기본 스플라인을 그립니다.

```csharp
public void DrawClosedCurve(Pen pen, PointF[] points, float tension, FillMode fillmode)
```

| 모수 | 유형 | 설명 |
| --- | --- | --- |
| pen | Pen | Pen 곡선의 색상, 너비 및 높이를 결정합니다. |
| points | PointF[] | 의 배열PointF 스플라인을 정의하는 구조. |
| tension | Single | 곡선의 장력을 지정하는 0.0F 이상의 값. |
| fillmode | FillMode | 곡선을 채우는 방법을 결정하는 FillMode 열거형의 멤버입니다. 이 매개 변수는 필수이지만 무시됩니다. |
