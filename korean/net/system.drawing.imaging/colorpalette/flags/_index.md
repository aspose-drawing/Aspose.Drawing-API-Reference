## ColorPalette.Flags property

색상 배열에서 색상 정보를 해석하는 방법을 지정하는 값을 가져옵니다.

```csharp
public int Flags { get; }
```

### 반환 값

다음 플래그 값이 유효합니다. 0x00000001 배열의 색상 값은 알파 정보를 포함합니다. 0x00000002 배열의 색상은 회색조 값입니다. 0x00000004 배열의 색상은 하프톤 값입니다.
