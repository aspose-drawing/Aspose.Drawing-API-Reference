## DrawingSettings.StrictMode property

获取或设置一个值，该值指示是否启用更严格的未实现功能捕获。

```csharp
public static bool StrictMode { get; set; }
```

### 评论

如果设置为 true，如果使用在当前实现中无法正常工作的功能/参数 库将抛出 NotImplementedException。 如果设置为 false，可能会忽略某些参数以使用户程序有机会工作。 在这种情况下与 GDI+. 相比，绘图结果可能看起来不同
