## MetafileType enumeration

指定元文件的类型。

```csharp
public enum MetafileType
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| Invalid | `0` | 指定在 GDI+ 中无法识别的元文件格式。 |
| Wmf | `1` | 指定 WMF（Windows 元文件）文件。这样的文件只包含 GDI 记录。 |
| WmfPlaceable | `2` | 指定一个 WMF（Windows 元文件）文件，它前面有一个可放置的元文件头。 |
| Emf | `3` | 指定增强元文件 (EMF) 文件。这样的文件只包含 GDI 记录。 |
| EmfPlusOnly | `4` | 指定 EMF+ 文件。此类文件仅包含 GDI+ 记录，必须使用 GDI+ 显示。 使用 GDI 显示记录可能会导致不可预知的结果。 |
| EmfPlusDual | `5` | 指定 EMF+ Dual 文件。此类文件包含 GDI+ 记录以及替代 GDI 记录，可以使用 GDI 或 GDI+ 显示。 使用 GDI 显示记录可能会导致质量下降。 |
