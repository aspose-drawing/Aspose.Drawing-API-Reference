## ImageLockMode enumeration

指定传递给[`LockBits`](../../system.drawing/bitmap/lockbits)方法. [`LockBits`](../../system.drawing/bitmap/lockbits)方法锁定 image 的一部分，以便您可以读取或写入像素数据。

```csharp
public enum ImageLockMode
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| ReadOnly | `1` | 指定图像的一部分被锁定以供阅读。 |
| WriteOnly | `2` | 指定图像的一部分被锁定以供写入。 |
| ReadWrite | `3` | 指定图像的一部分被锁定以供读取或写入。 |
| UserInputBuffer | `4` | 指定用于读取或写入像素数据的缓冲区由用户分配。 如果设置了此标志，则*flags*的参数[`LockBits`](../../system.drawing/bitmap/lockbits) 方法用作输入参数（也可能用作输出参数）。 如果清除此标志，则*flags*参数仅用作输出参数。 |
