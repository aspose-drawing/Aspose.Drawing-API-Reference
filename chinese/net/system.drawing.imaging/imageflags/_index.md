## ImageFlags enumeration

指定包含在一个像素数据的属性[`Image`](../../system.drawing/image)目的。 Flags 属性返回此枚举的成员。 此枚举有一个 FlagsAttribute 属性，允许按位组合其成员值。

```csharp
[Flags]
public enum ImageFlags
```

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| None | `0` | 没有格式信息 |
| Scalable | `1` | 像素数据是可缩放的。 |
