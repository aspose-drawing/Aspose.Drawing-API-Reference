## AddClosedCurve(PointF[], float) {#addclosedcurve_1}

Добавляет к этому пути замкнутую кривую. Используется кардинальная кривая сплайна, поскольку кривая проходит через каждую точку массива.

```csharp
public void AddClosedCurve(PointF[] points, float tension)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| points | PointF[] | Массив структур PointF, представляющий точки, определяющие кривую. |
| tension | Single | Значение от 0 до 1, указывающее величину изгиба кривой между точками, где 0 — наименьшая кривая (самый острый угол), а 1 — самая плавная кривая. |
