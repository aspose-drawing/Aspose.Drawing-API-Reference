Указывает, как можно комбинировать различные области отсечения.

```csharp
public enum CombineMode
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| Replace | `0` | Одна область отсечения заменена другой. |
| Intersect | `1` | Две области отсечения объединяются путем их пересечения. |
| Union | `2` | Две области отсечения объединяются путем объединения обоих. |
| Xor | `3` | Два региона отсечения объединяются, беря только области, заключенные в один или другой регион, но не в оба. |
| Exclude | `4` | Указывает, что существующий регион заменяется результатом удаления нового региона из существующего региона. Иными словами, новый регион исключается из существующего региона. |
| Complement | `5` | Указывает, что существующий регион заменяется результатом удаления существующего региона из нового региона. Иными словами, существующий регион исключается из нового региона. |
