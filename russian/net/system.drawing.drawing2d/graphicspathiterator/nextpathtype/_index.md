## GraphicsPathIterator.NextPathType method

Получает начальный индекс и конечный индекс следующей группы точек данных, имеющих один и тот же тип.

```csharp
public int NextPathType(out byte pathType, out int startIndex, out int endIndex)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| pathType | Byte& | [out] Получает тип точки, общий для всех точек в группе. Возможные типы можно получить изPathPointType перечисление. |
| startIndex | Int32& | [out] Получает начальный индекс группы точек. |
| endIndex | Int32& | [out] Получает конечный индекс группы точек. |

### Возвращаемое значение

Этот метод возвращает количество точек данных в группе. Если в пути больше нет групп, этот метод возвращает 0.
