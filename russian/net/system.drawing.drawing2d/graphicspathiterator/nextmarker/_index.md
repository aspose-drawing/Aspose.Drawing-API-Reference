## NextMarker(out int, out int) {#nextmarker}

Увеличивает[`GraphicsPathIterator`](../../graphicspathiterator)к следующему маркеру в path и возвращает начальный и конечный индексы посредством параметров [out].

```csharp
public int NextMarker(out int startIndex, out int endIndex)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| startIndex | Int32& | [out] Целочисленная ссылка, указанная в этом параметре, получает индекс точки, с которой начинается подпуть. |
| endIndex | Int32& | [out] Целочисленная ссылка, предоставленная этому параметру, получает индекс точки, которая заканчивает подпуть, на который указывает startIndex. |

### Возвращаемое значение

Количество точек между этим маркером и следующим.

## NextMarker(GraphicsPath) {#nextmarker_1}

Это[`GraphicsPathIterator`](../../graphicspathiterator) объект имеет[`GraphicsPath`](../../graphicspath) объект, связанный с ним. Этот метод увеличивает связанный с ним[`GraphicsPath`](../../graphicspath) к следующему маркеру в своем пути и копирует все точки, содержащиеся между текущим маркером и следующим маркером (или концом пути) во второй[`GraphicsPath`](../../graphicspath) объект, переданный в параметр.

```csharp
public int NextMarker(GraphicsPath path)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| path | GraphicsPath | [`GraphicsPath`](../../graphicspath) объекта, на который будут скопированы точки. |

### Возвращаемое значение

Количество точек между этим маркером и следующим.
