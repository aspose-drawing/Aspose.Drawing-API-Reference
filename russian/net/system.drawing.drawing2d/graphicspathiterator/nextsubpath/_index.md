## NextSubpath(out int, out int, out bool) {#nextsubpath}

Перемещает[`GraphicsPathIterator`](../../graphicspathiterator) к следующему подпути в пути. Начальный индекс и конечный индекс следующего подпути содержатся в параметрах [out].

```csharp
public int NextSubpath(out int startIndex, out int endIndex, out bool isClosed)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| startIndex | Int32& | [out] Получает начальный индекс следующего подпути. |
| endIndex | Int32& | [out] Получает конечный индекс следующего подпути. |
| isClosed | Boolean& | [out] Указывает, закрыт ли подпуть. |

### Возвращаемое значение

Количество точек данных в извлеченной фигуре (подпути). Если больше нет фигур для извлечения, возвращается ноль.
