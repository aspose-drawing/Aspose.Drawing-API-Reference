Определяет качество рендеринга текста.

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| SystemDefault | `0` | Каждый символ рисуется с использованием своего растрового изображения глифа с системной подсказкой по умолчанию. Текст будет отображаться с использованием любых настроек сглаживания шрифта, выбранных пользователем для системы. |
| SingleBitPerPixelGridFit | `1` | Каждый символ рисуется с использованием своего растрового изображения глифа. Подсказка используется для улучшения внешнего вида символов на основе и кривизне. |
| SingleBitPerPixel | `2` | Каждый символ рисуется с использованием своего растрового изображения глифа. Подсказка не используется. |
| AntiAliasGridFit | `3` | Каждый символ рисуется с использованием сглаженного растрового изображения глифа с хинтингом. Гораздо лучшее качество благодаря сглаживанию, но при более высокой производительности. |
| AntiAlias | `4` | Каждый символ рисуется с использованием сглаженного растрового изображения глифа без хинтинга. Лучшее качество благодаря сглаживанию. Различия в ширине стержня могут быть заметны, поскольку хинтинг отключен. |
| ClearTypeGridFit | `5` | Каждый символ рисуется с использованием растрового изображения ClearType глифа с хинтингом. Настройка наивысшего качества. Используется для использования преимуществ шрифта ClearType. |
