## Pen.CompoundArray property

Получает или задает массив значений, указывающий составное перо. Составное перо рисует составную линию, состоящую из параллельных линий и пробелов.

```csharp
public float[] CompoundArray { get; set; }
```

### Возвращаемое значение

Массив вещественных чисел, задающий составной массив. Элементы массива должны быть в порядке возрастания, не меньше 0 и не больше 1.
