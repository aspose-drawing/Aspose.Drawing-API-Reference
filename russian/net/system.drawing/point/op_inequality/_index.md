## Point Inequality operator

Сравнивает дваPoint objects. Результат указывает, будут ли значенияX или жеY properties из двухPoint объекты не равны.


| Параметр | Тип | Описание |

| left | Point | ПервыйPoint сравнивать. |
| right | Point | ВторойPoint сравнивать. |

### Возвращаемое значение
