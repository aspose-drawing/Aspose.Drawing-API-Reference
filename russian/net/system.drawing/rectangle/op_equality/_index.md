## Rectangle Equality operator

Проверяет, являются ли дваRectangle структуры имеют одинаковое расположение и размер.

```csharp
public static bool operator ==(Rectangle left, Rectangle right)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| left | Rectangle | Rectangle структура слева от оператора равенства. |
| right | Rectangle | Rectangle структуру, которая находится справа от оператора равенства. |

### Возвращаемое значение

Этот оператор возвращает`истинный` если дваRectangle структуры имеют одинаковые свойства X, Y, Width и Height.
