## DrawingSettings.StrictMode property

Получает или задает значение, указывающее, включен ли более строгий отлов нереализованных функций.

```csharp
public static bool StrictMode { get; set; }
```

### Примечания

Если установлено значение true, в случае использования функций/параметров, которые не работают корректно в текущей реализации , библиотека выдаст исключение NotImplementedException. Если установлено значение false, некоторые параметры могут быть проигнорированы, чтобы дать возможность пользовательской программе работать. В этом случае результаты рисования могут отличаться от GDI+.
