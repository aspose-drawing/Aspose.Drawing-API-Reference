## Graphics.DrawIconUnstretched method

Рисует изображение, представленное указаннымIcon без масштабирования изображения.

```csharp
public void DrawIconUnstretched(Icon icon, Rectangle targetRect)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| icon | Icon | Icon рисовать. |
| targetRect | Rectangle | Rectangle структура, указывающая расположение и размер результирующего изображения. Изображение не масштабируется, чтобы соответствовать этому прямоугольнику, но сохраняет свой первоначальный размер. Если изображение больше прямоугольника, оно обрезается, чтобы поместиться внутри него. |
