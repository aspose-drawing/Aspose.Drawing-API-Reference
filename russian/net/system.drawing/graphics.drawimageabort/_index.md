## Graphics.DrawImageAbort delegate

Предоставляет метод обратного вызова для принятия решения о том, когда[`DrawImage`](../graphics/drawimage) метод должен преждевременно отменить выполнение и прекратить отрисовку изображения.

```csharp
public delegate bool DrawImageAbort(IntPtr callbackdata);
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| callbackdata | IntPtr | Внутренний указатель, указывающий данные для метода обратного вызова. Этот параметр передается не всем[`DrawImage`](../graphics/drawimage) перегрузки. Вы можете проверить его отсутствие, проверив значениеZero . |

### Возвращаемое значение

Этот метод возвращает значение true, если он решает, что[`DrawImage`](../graphics/drawimage) метод должен преждевременно остановить выполнение. В противном случае он возвращает false, чтобы указать, что[`DrawImage`](../graphics/drawimage) метод должен продолжать выполнение.
