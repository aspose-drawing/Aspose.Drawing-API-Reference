## MetafileType enumeration

Определяет типы метафайлов.

```csharp
public enum MetafileType
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| Invalid | `0` | Указывает формат метафайла, который не распознается в GDI+. |
| Wmf | `1` | Задает файл WMF (метафайл Windows). Такой файл содержит только записи GDI. |
| WmfPlaceable | `2` | Указывает файл WMF (метафайл Windows), перед которым помещается заголовок метафайла. |
| Emf | `3` | Указывает файл расширенного метафайла (EMF). Такой файл содержит только записи GDI. |
| EmfPlusOnly | `4` | Указывает файл EMF+. Такой файл содержит только записи GDI+ и должен отображаться с помощью GDI+. Отображение записей с помощью GDI может привести к непредсказуемым результатам. |
| EmfPlusDual | `5` | Указывает двойной файл EMF+. Такой файл содержит записи GDI+ вместе с альтернативными записями GDI и может отображаться с помощью GDI или GDI+. Отображение записей с помощью GDI может привести к некоторому ухудшению качества. |
