## StringFormat.DigitSubstitutionLanguage property

Hämtar språket som används när lokala siffror ersätts med västerländska siffror.

```csharp
public int DigitSubstitutionLanguage { get; }
```

### Returvärde

En språkidentifierare för National Language Support (NLS) som identifierar språket som kommer att användas när lokala siffror ersätts med västerländska siffror. Du kan passeraLCID egendom hos enCultureInfo objekt som NLS språkidentifierare. Anta till exempel att du skapar enCultureInfoobjekt genom att skicka strängen "ar-EG" till aCultureInfo konstruktör. Om du passerarLCID egenskap av detCultureInfo objekt tillsammans med.Traditional to den[`SetDigitSubstitution`](../setdigitsubstitution) metod, då kommer arabisk-indiska siffror att ersätta västerländska siffror vid visningstidpunkten.
