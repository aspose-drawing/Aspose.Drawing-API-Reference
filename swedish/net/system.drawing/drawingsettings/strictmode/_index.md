## DrawingSettings.StrictMode property

Hämtar eller ställer in ett värde som indikerar om den mer strikta fångsten av icke implementerade funktioner är aktiverad.

```csharp
public static bool StrictMode { get; set; }
```

### Anmärkningar

Om det är satt till true, om funktioner/parametrar som inte fungerar korrekt i nuvarande implementation kommer biblioteket att kasta NotImplementedException. Om satt till false kan vissa parametrar ignoreras för att ge användarprogrammet chansen att fungera. I så fall ritningsresultat kan se annorlunda ut jämfört med GDI+.
